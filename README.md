Project Overview

This project focuses on analyzing agricultural irrigation data using Python and Data Science techniques. The goal is to understand irrigation patterns across different states and build a predictive model to estimate Gross Irrigated Area.

🎯 Objectives
Analyze irrigation patterns across states and years
Compare different irrigation sources (Canal, Tubewell, Tank, Others)
Perform Exploratory Data Analysis (EDA)
Visualize trends using graphs and plots
Build a Machine Learning model for prediction
🛠️ Technologies Used
Python
Pandas
NumPy
Matplotlib
Seaborn
Scikit-learn
📊 Key Features
Data Cleaning & Preprocessing
Exploratory Data Analysis (EDA)
Data Visualization (Line plot, Pie chart, Heatmap, Scatter plot, Boxplot)
Statistical Analysis
Linear Regression Model
📈 Insights
Tubewell irrigation contributes the largest share
Strong positive relationship between Net & Gross Irrigated Area
Significant variation across different states
Presence of skewness and outliers in data
🤖 Machine Learning
Model Used: Linear Regression
R² Score: ~0.85
Successfully predicted Gross Irrigated Area
📂 Dataset
Source: National Data & Analytics Platform (NDAP), NITI Aayog
Link: https://ndap.niti.gov.in/dataset/6796
